# 旅游信息管理系统 python290

## 项目概述

旅游信息管理系统是基于Python与Vue开发的，集成了Django框架和Scrapy爬虫技术，致力于提供全面的旅游信息推荐和管理服务。

## 技术栈

- 前端：Vue
- 后端：Python、Django、Scrapy爬虫
- 数据库：MySQL

## 功能模块

### 用户模块

- 登录
- 注册
- 首页
- 景点信息
- 热门景点信息
- 特色美食
- 游玩路线
- 个人中心
- 投诉建议
- 旅游资讯

### 管理员模块

- 用户管理
- 景点信息管理
- 热门景点管理
- 特色美食管理
- 游玩路线管理
- 基础数据类型管理
- 轮播图管理
- 个人中心
- 资讯管理

### 员工模块

- 活动查看
- 个人中心
- 订单管理

## 系统角色

- 用户
- 管理员
- 员工

## 运行环境

- 后端：Python 3.x，Django 2.x，Scrapy 1.x
- 前端：Vue.js，具体版本依赖请参考项目文档
- 数据库：MySQL 5.x 或更高版本
